1) Purchase Module
------------------

1. /Purchase Order page

** View All created POs in table

** Create POs
 - Select Vendor (Search & Select Dropdown)
 - Select Category (Autoselected)
 - Select Item Type (Autoselected)
 - PO Date (Select Date)
 - Delivery Date (Select Date)
 - Select Item Price History (All Vendors | Current Vendor)

 - Add Items
  > Item name (Need to Search & Select Dropdown)
  > Pkg Size (Autoselected)
  > Item code (Autoselected)
  > HSN Code (Optional)
  > In Stock (Auto-fetched)
  > Quantity (Need to type)
  > Unit Cost (Autogenerated & Editable)
  > Tax Rate (Optional)
  > Total Price (Auto-calculated)
  > Add Description (Optional)

 - View Added Items in Table
 - Create PO (button)

** After creating PO, navigate to /Generate GRN page
 - Buttons on top
   > Back
   > Export
   > Detailed Print
   > Print
   > Send PO Copy
   > Save
   > "Generate GRN" button - final
 - PO No (Autogenerated)
 - PR No (Autogenerated)
 - PO Created Date (Autogenerated)
 - Status (Default - open)
 - Receiving Branch (Autogenerated)
 - Vendor (Autogenerated)

 - Work Area (Dropdown - Need to select) Ex: Store, Main Kitchen, etc.,
 - Goods Receive Date (select)
 - Vendor Invoice Date (select)
 - Invoice Number (Need to type)
 
 - View all PO items in table format

-----------------------------------------------------------------------

2. /Special Order page (Same as Purchase Order, only used if one-time new vendor PO / immediate PO)

** View All created SOs in table

** Create SOs

** Generate GRN

-----------------------------------------------------------------------

3. /GRNS page - {Used to track items, quantity, and Work Area location}

** View All generated GRNs in table
   > GRN ID (button) - User can able to access GRN ID page
   > Request ID
   > Invoice ID
   > Vendor Name
   > Total Amount
   > GRN Date
   > Vendor Invoice ID
   > Action (button) - User can able to update RTV (Return quantity for each item in that GRN)

** After clicking on Action button in any row of table - /GRN ID RTV page will open
 - User can able to view all items in table and can edit Quantity
 - Then click "Generate RTV" button to update
 - It should increment/deduct based on updated Quantity value to inventory

** After clicking on any GRN ID from table row - /GRN ID page will open
 - Buttons on top
   > Back
   > Upload Invoice
   > Export
   > Print
   > Save

 - View Static GRN details card
   > GRN ID
   > Order Number
   > Recipient Number
   > Invoice ID (editable)
   > Good Received Date
   > Sender Name
   > GRN Date (System-generated)
   > Vendor Invoice Date (editable)

 - Preview GRN items in table format with below columns
   > S.No
   > Item code
   > Item name
   > Pkg Name
   > Received Quantity
   > In Store
   > RTV Qty (returned quantity - initially 0)
   > Unit Cost
   > Tax Rate
   > Extra Amt
   > Discount
   > Cess Amt
   > Tax Amt
   > Total
   {> Main_Kitchen Qty
   > Store Qty
   > etc., (if more Work Area applicable)} - Dynamic columns
